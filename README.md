# tbv_medical_hallucinations_auditing_mafl
Codebase for the paper “Trust but Verify: Mitigating Medical Hallucinations via Post-Hoc Adversarial Auditing and Multi-Agent Feedback Loops” (arXiv:2606.14149). Implements a multi-agent LLM framework for detecting and reducing hallucinations in medical QA systems using adversarial auditing and feedback-driven verification mechanisms.
